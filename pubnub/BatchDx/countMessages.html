Future<CountMessagesResult> countMessages (
  1. dynamic channels,
  2. {Keyset keyset,
  3. String using,
  4. Timetoken timetoken}
)

Get multiple channels' message count using one call.

channels can either be a Map<String, Timetoken> or Set<String>:

  • if you want to count messages in all channels up to a common timetoken, pass in a Set<String> and a named parameter timetoken.
  • if you want to specify separate timetoken for each channel, pass in a Map<String, Timetoken>. Additionally, if a value in the map is null, it will use a timetoken from a named parameter timetoken.
